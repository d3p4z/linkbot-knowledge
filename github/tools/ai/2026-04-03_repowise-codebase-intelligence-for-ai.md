# Repowise - Codebase Intelligence for AI

## What This Is
Repowise provides codebase intelligence for AI-assisted engineering teams. It auto-generates documentation, analyzes git history, detects dead code, and captures architectural decisions, making codebases understandable to AI agents.

## AppSec Relevance
Repowise enhances application security by surfacing code risks like hotspots, hidden coupling, and stale architectural decisions that can lead to vulnerabilities. It helps security engineers and developers understand the 'why' behind code, reducing the likelihood of introducing security flaws.

- Git history analysis (hotspots, ownership, co-changes)
- AI-generated documentation with freshness scoring
- Architectural decision tracking and staleness monitoring
- Dead code detection
- MCP server for AI agent integration
