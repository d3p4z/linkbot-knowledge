# AndroHunter

## What This Is
A native Android security toolkit built with Jetpack Compose that allows researchers to perform static and dynamic analysis of apps directly on the device.

## AppSec Relevance
It provides a comprehensive suite for testing Android vulnerabilities such as intent hijacking, SQL injection in content providers, path traversal, and insecure data storage without requiring a full desktop setup.

## Key Capabilities
- On-device DEX and Manifest analysis for hardcoded secrets and exported components
- Automated fuzzing for Intents, Broadcasts, and Content Providers
- Integrated HTTP traffic interceptor and Frida script generator
- Specialized modules for FileProvider path traversal and Task Hijacking (StrandHogg) testing
