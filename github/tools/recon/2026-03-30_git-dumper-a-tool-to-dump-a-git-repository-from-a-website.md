# git-dumper: A tool to dump a git repository from a website

## What This Is
This tool downloads a full git repository from a web server, even if directory listing is disabled, by intelligently discovering and fetching git objects and references.

## AppSec Relevance
AppSec engineers can use this tool to identify if sensitive source code is exposed via misconfigured web servers, which could lead to critical vulnerabilities through source code analysis.

## Key Capabilities
- Dump a git repository from a website
- Works even when directory listing is disabled
- Intelligently discovers git refs, objects, and commits
- Reconstructs the full working tree
- Supports proxies, custom user-agents, and HTTP headers
- Client certificate support (PKCS#12)

## Reach For This When
When performing external black-box assessments or bug bounty hunting to check for exposed `.git` directories and potentially leak source code due to server misconfiguration.
